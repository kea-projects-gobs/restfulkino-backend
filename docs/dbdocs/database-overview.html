Cinema

Attributter:

  • id: int
  • name: string
  • city: string
  • street: string
  • description: string
  • phone: string
  • email: string
  • imageUrl: string
  • isActive: boolean
  • halls: Hall (relation)

Relationer:

  • Hall: one-to-many - En biograf kan have flere sale

Hall

Attributter:

  • id: int
  • name: string
  • noOfRows: int
  • noOfColumns: int
  • imageUrl: string
  • isActive: boolean
  • cinemas: Cinema (relation)
  • schedules: Schedule (relation)
  • seats: Seat (relation)

Relationer:

  • Cinema: many-to-one - En sal er en del af en specifik biograf.
  • Schedule: one-to-many - En sal kan have flere visninger.
  • Seat: one-to-many - En sal kan have flere sæder.

Movie

Attributter:

  • id: int
  • title: string
  • description: string
  • releaseDate: LocalDate
  • duration: int
  • imageUrl: string
  • language: string
  • genre: string
  • director: string
  • cast: string
  • isActive: boolean
  • schedule: Schedule (relation)

Relationer:

  • Schedule: one-to-many - En film kan have flere visninger.

MoviePrice

Attributter:

  • name: string (ID)
  • amount: double
  • unit: string

Reservation

Attributter:

  • id: int
  • schedule: Schedule (relation)
  • tickets: Ticket (relation)
  • reservationDate: LocalDate
  • feeOrDiscount: double
  • user: UserWithRoles (relation)

Relationer:

  • Schedule: many-to-one - En reservation er knyttet til en specifik forestilling.
  • Tickets: one-to-many - En reservation kan omfatte flere billetter.
  • UserWithRoles: many-to-one - En reservation er knyttet til en specifik bruger, der har foretaget bookingen.

ReservationPrice

Attributter:

  • name: string (ID)
  • amount: double
  • unit: string

ReservationPrice repræsenterer prisen for en reservation.

Schedule

Attributter:

  • id: Integer
  • startTime: LocalTime
  • endTime: LocalTime
  • date: LocalDate
  • is3d: boolean
  • isLongMovie: boolean
  • movie: Movie (relation)
  • hall: Hall (relation)

Schedule repræsenterer en specifik filmvisning i en biograf.

Relationer:

  • Movie: many-to-one - En visning er knyttet til en specifik film.
  • Hall: many-to-one - En visning finder sted i en specifik sal.

Seat

Attributter:

  • id: int
  • seatIndex: int
  • seatPrice: SeatPrice (relation)
  • hall: Hall (relation)
  • isActive: boolean

Seat repræsenterer et individuelt sæde i en biografsal.

Relationer:

  • SeatPrice: many-to-one - Et sæde kan have en specifik pris, der er defineret af en SeatPrice.
  • Hall: many-to-one - Et sæde er placeret i en specifik sal.

SeatPrice

Attributter:

  • name: string (ID)
  • amount: double
  • unit: string

SeatPrice repræsenterer prisen for et sæde

Ticket

Attributter:

  • id: int
  • seat: Seat (relation)
  • price: double
  • reservation: Reservation (relation)

Ticket repræsenterer en billet til en filmvisning.

Relationer:

  • Seat: many-to-one - En billet er knyttet til et specifikt sæde.
  • Reservation: many-to-one - En billet er en del af en specifik reservation.
